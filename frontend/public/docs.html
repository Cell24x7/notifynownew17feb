Overview

NotifyNow is an omni-channel messaging automation platform designed for enterprises and resellers. It integrates WhatsApp, RCS, SMS, Voice, and Email into a single dashboard with advanced failover capabilities.

Reseller Ready

Manage multiple clients, set custom pricing, and monitor usage pools.

Developer API

Robust REST APIs for transactional OTPs and automated notifications.

Admin: Creating a New User

1

Access Clients Panel

Navigate to Super Admin > Clients from the sidebar.

2

Fill Details

Click on + Add Client. Enter Name, Email, Password, and Mobile Number.

3

Role Selection

Choose Client (Direct End User) or Reseller (Who can create their own clients).

Roles & Permissions

Every user can have specific channels enabled or disabled. This controls their dashboard menu and API access.

  • WhatsApp Access
  • RCS Access
  • SMS Access
  • Voice & Email Access

Credit Management

NotifyNow works on a pre-paid wallet model. Users must have balance to send campaigns.

Methods to Add Credits:

  1. Manual Adjustment (Admin): Go to Wallet > Adjust, select the user, choose 'Credit', enter amount, and hit save.
  2. Self-Recharge (User): Users can click "Top-up Wallet" on their dashboard and pay via CCAvenue.

Gateway Assignment

Gateways define which provider is used to route messages. This is crucial for pricing and deliverability.

SMS Gateway

Go to Integrations > SMS Gateways. Assign a specific gateway to a user. This gateway must have their DLT-approved Sender ID.

RCS Config

Enable specific Bots for users in Integrations > RCS Config.

WhatsApp Business API

Requires a Meta-approved Business Account and WhatsApp Number.

  • Template Creation: Must include Variables like {{1}}, {{2}} for dynamic content.
  • Approval: Meta takes 2 mins to 2 hours to approve templates.

Static vs Dynamic Campaigns

Static Campaigns

The same message is sent to all selected contacts. Best for general announcements or festival greetings.

Dynamic (CSV) Campaigns - The Powerhouse

Send personalized messages to thousands by uploading a CSV/Excel file. Each column in your file can be mapped to a variable in your template.

Example Template: Dear {{1}}, your order ID {{2}} is ready!

Your CSV should have columns like 'Name' and 'OrderID'. During step 3 of campaign creation, map {{1}} to 'Name' and {{2}} to 'OrderID'.

Smart Failover (Fallback)

Ensure 100% delivery. If a WhatsApp or RCS message fails (e.g., user is offline), the system automatically sends an SMS.

How to Setup?
  1. During Campaign Creation, enable "Failover".
  2. Select an SMS Template (DLT approved).
  3. The system will automatically map dynamic variables from WhatsApp/RCS to the SMS template.
